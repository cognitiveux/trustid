Βήμα 2: Εκκίνηση Web κάμερας και ελέγχων

Σε αυτό το βήμα θα σας ζητηθεί να ενεργοποιήσετε την Web κάμερα σας και το TRUSTID θα πραγματοποιήσει διάφορους ελέγχους. Σε αυτό το σημείο, δεν θα καταγραφούν ακόμη δεδομένα προσώπου. Η καταγραφή θα ξεκινήσει μόλις νιώσετε άνετα και το επιθυμείτε στο Βήμα 3.


Βήμα 3: Καταγραφή δεδομένων προσώπου

Αρχικά, τοποθετήστε το πρόσωπό σας στο πλαίσιο της κάμερας σε μια απόσταση που σας βολεύει. Στη συνέχεια, θα ακολουθήσετε 5 βήματα στα οποία θα μετακινήσετε το πρόσωπό σας προς όλες τις κατευθύνσεις - αριστερά (δεξί μάγουλο στραμμένο προς κάμερα), δεξιά (αριστερό μάγουλο στραμμένο προς κάμερα), μπροστά, πάνω, κάτω.

Σε κάθε πόζα, κάντε κλικ στο αντίστοιχο κουμπί και μιμηθείτε τις κινήσεις όπως θα φαίνονται στο σύντομο animation που θα εμφανιστεί.

Ασφαλής επεξεργασία και μεταφόρτωση δεδομένων προσώπου.
Μιμηθείτε επανειλημμένα την κίνηση μέχρι να λάβετε ανατροφοδότηση.
Κάθε στάση του κεφαλιού διαρκεί περίπου 60 δευτερόλεπτα.

Σας ευχαριστούμε για τη συμμετοχή σας στην πρώτη φάση της μελέτης χρηστών του TRUSTID

Τα δεδομένα σας έχουν αποθηκευτεί με επιτυχία. Τις επόμενες ημέρες θα σας αποσταλεί ένα email για να επισκεφθείτε την ίδια σελίδα και να ακολουθήσετε τα ίδια βήματα που κάνατε και τώρα.

Τα δεδομένα θα είναι εμπιστευτικά και θα επεξεργάζονται και θα αποθηκεύονται με ασφάλεια καθ' όλη τη διάρκεια του έργου. Στο τέλος του έργου, όλα τα δεδομένα θα διαγραφούν οριστικά από τη βάση δεδομένων του έργου. Πρόσβαση στα δεδομένα θα έχουν μόνο ερευνητές του έργου και θα χρησιμοποιηθούν μόνο για ερευνητικούς σκοπούς.

Σε περίπτωση που επιθυμείτε να διαγραφούν τα δεδομένα σας, παρακαλούμε όπως αποστείλετε ένα μήνυμα στο unscubscribe_poc1@trustid-project.eu χρησιμοποιώντας το ίδιο email που χρησιμοποιήσατε κατά την εγγραφή σας στη μελέτη χρηστών και τότε όλα τα δεδομένα σας θα διαγραφούν οριστικά από τη βάση δεδομένων του έργου.
